# Dima Tkachuk — QA Automation Portfolio

This is my portfolio showcasing manual and automation testing projects using Python, Pytest, Playwright, and Selenium for web UI testing, along with Python Requests and Postman for API testing.

All projects have GitHub Actions workflows for automated test runs.

---

## Main Projects

- `automation_playwright` — Web UI automation using Playwright + Pytest (Page Object Model + GitHub Actions)
- `automation_selenium` — Web UI automation using Selenium + Pytest (Page Object Model + GitHub Actions)
- `api_testing` — REST API testing with Python Requests and Postman (CRUD operations, status codes, JSON validation + GitHub Actions)
- `website_for_automation_testing` — Custom website I built for practicing automation (forms, iframes, links, etc.)
- `oop_python` — OOP and test framework foundations (with GitHub Actions)
