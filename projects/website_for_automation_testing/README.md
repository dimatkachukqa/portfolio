# Website for Automation Testing

This is a custom website I built specifically for practicing web UI automation with Playwright and Selenium.

It includes multiple pages with forms, links, iframes, and various elements to test locators, interactions, and synchronization.

## Running the Project

1. Navigate to `projects/website_for_automation_testing/src`
2. Open the homepage `index.html` in your browser
3. Use the links to navigate between the homepage, form page, and iframe page.
